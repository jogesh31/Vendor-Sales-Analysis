Distributor Sales Analytics (Power BI Dashboard)

üìå Project Overview

This project focuses on analyzing distributor-level sales performance for Aspen Surgical over the previous 12 months. The main objective was to help the Stakeholder understand extended cost trends, distributor contribution, geographic performance, and top items/customers for strategic decision-making.

The analysis was performed using Power BI, leveraging advanced data modeling, DAX measures, conditional formatting, and interactive visualizations.


üéØ Stakeholder Requirement

The Stakeholder wanted a comprehensive dashboard that could:

Analyze sales performance by distributors under different vendor codes.

Track top-selling items based on extended cost.

Identify high-value customers (end users) by facility, state, and city.

Understand geographic performance across the U.S.

Compare monthly invoice trends, seasonality, and sales spikes.

Allow users to drill down using slicers (invoice date, vendor code, distributor, and item code).
